# HR & Payroll Management System - Frontend (React)

## Overview
This is the **frontend for the HR & Payroll Management System**. It provides an intuitive, responsive dashboard for admins to manage employees, payroll, leave requests, and reporting. The frontend communicates with the Django backend via REST APIs.

## Features
- Employee management dashboard
- Payroll validation and PDF generation
- Leave request approvals
- Responsive and user-friendly interface
- API integration with Django backend

## Technology Stack
- React
- Axios (for API calls)
- Tailwind CSS (optional for styling)

## Installation
1. Clone the repository:
```bash
git clone https://github.com/DanielMbe/HR-Payroll-Management-System-React-Frontend.git
cd gestion_rh_paie_ui
